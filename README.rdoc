== CLASEB BACKEND

Backend REST de recolección y muestra de estadisticas para ClaseB (Simulador de conducción en realidad virtual utilizando Oculus, Unity3D y Logitch G27.)


Para utilizar el proyecto es necesario:

* bundle install

* rake db:create

* rails s

El deployment solia estar configurado utilizando capistrano en conjunto con un server hosteado en DigitalOcean que fue apagado luego de finalizar el proyecto.
